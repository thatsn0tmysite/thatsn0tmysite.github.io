XSServe

I promise I will revise this page with a decent description of the project, a roadmap, possibly some documentation and stuff. For now, here is the bare bone README.md copy pasted for your own convenience.

XSServe is a shameless copy of heavely inspired by the XSSHunter by (@IAmMandatory) project, rewritten in Go.

üèÅ Goals

The initial goal is to allow users to use the same service, but in a self-contained way for lazy penetration testers, like myself. Currently I am using MongoDB as a backend, but I’d really love a serverless nosql database as a default option. I wasn’t able to find any.

The final goal is still unclear as the project might evolve as different needs arise.
